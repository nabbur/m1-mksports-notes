# m1-mksports-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为个人笔记、项目记录或功能演示页面的静态存放与展示空间。

## 目录说明

```
.
├── index.html          # 仓库根目录入口页面（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放独立的 HTML 页面，每个页面可单独通过链接访问。
- `assets/`：存放页面引用的样式、脚本与图片等资源文件，便于统一管理。

## 页面归档说明

- 所有 HTML 页面均为独立文件，不依赖后端服务，可直接在浏览器中打开。
- 页面内容涵盖个人学习笔记、功能演示、工具页面等，不涉及商业或营销用途。
- 页面可能引用外部资源（如公共 CDN），但不包含任何指向特定网站或域名的链接。

## 维护说明

- 本仓库采用静态页面形式，更新时直接替换或新增 HTML 文件即可。
- 建议保持 `pages/` 与 `assets/` 目录结构清晰，便于后续检索与复用。
- 如有需要，可在根目录 `index.html` 中添加页面导航或索引列表。
- 欢迎提交 Issue 或 Pull Request 进行内容补充或结构优化。
